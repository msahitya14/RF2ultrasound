# RF data to Ultrasound B mode images

This repository contains code for converting radio frequency echo data - in this case from 128 lines using a micro convex probe by interson. Code is given in ***main.py*** and ***finalise.ipynb*** as the notebook.

To install requirements in the python environment use:

```pip install -r requirements.txt```


Python version used in this project was:

```Python 3.13.3```
